# ☕ Dashboard and Analysis: Coffee Shop

## 📌 Descripción del Proyecto
Este proyecto realiza un análisis exploratorio de datos (EDA) y la creación de un dashboard interactivo enfocado en las ventas y el comportamiento de clientes de una cafetería (*Coffee Shop*). 

El objetivo principal es identificar patrones de compra, horas de mayor flujo, productos más vendidos y métricas clave para la toma de decisiones comerciales.

---

## 📊 Objetivos de Análisis
* **Ventas:** Tendencias de ingresos diarios, semanales y mensuales.
* **Productos:** Top de productos más vendidos y categorías con mayor margen.
* **Clientes:** Comportamiento de compra por horas pico y días de la semana.

---

## 🛠️ Herramientas Utilizadas
* **Lenguaje:** Python / SQL
* **Librerías de Análisis:** Pandas, NumPy
* **Visualización:** Power BI / Tableau / Matplotlib / Seaborn
* **Control de Versiones:** Git & GitHub
